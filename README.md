# vagrant-ubuntu14.04

This envirment is Ubuntu14.04 LTS use vagrant.  
You can launch the foundation of the web application in one word "vagrant up".  
It would be useful if you are using KGOYA's VPS.

```
vagrant up
-> http://192.168.33.101/index.php
```

* OS
    * Ubuntu 14.04 LTS
    * ubuntu/trusty64
* Web Server
    * Apache
* PHP
    * PHP5.6
* Database
    * MySQL5.6
